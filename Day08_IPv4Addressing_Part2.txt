**ðŸ“Œ Address Types**

- **Network Address:** All host bits = 0 (e.g., 192.168.1.0)
- **Broadcast Address:** All host bits = 1 (e.g., 192.168.1.255)
- **First Usable Address:** Network + 1 (e.g., 192.168.1.1)
- **Last Usable Address:** Broadcast - 1 (e.g., 192.168.1.254)
