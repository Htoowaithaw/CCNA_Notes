### ‚úÖ **Step 1 ‚Äì Configure R1 as DHCP Server**

---

### ‚úÖ **Step 2 ‚Äì Enable DHCP Snooping on SW1**

```bash
bash
CopyEdit
enable
conf t

# Enable globally
ip dhcp snooping

# Enable for VLAN 1
ip dhcp snooping vlan 1

# Trust G0/2 (interface to DHCP server)
interface g0/2
 ip dhcp snooping trust
exit

```

---

### ‚úÖ **Step 3 ‚Äì Enable DHCP Snooping on SW2**

```bash
bash
CopyEdit
enable
conf t

ip dhcp snooping
ip dhcp snooping vlan 1

# Trust uplink to SW1
interface g0/1
 ip dhcp snooping trust
exit

```
